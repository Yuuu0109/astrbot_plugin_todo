# 更新日志

## v1.0.0 (2026-02-17)

🎉 **首次发布**

### 新增功能

- ✨ 待办事项管理（添加 / 查看 / 完成 / 删除）
- ✨ 中文自然语言时间解析（明天下午三点、后天、3天后、下周一 等）
- ✨ 截止时间提醒（仅私聊，可配置提前时间）
- ✨ 逾期待办检查与提醒（仅私聊）
- ✨ 每日早报推送（仅私聊，可配置推送时间）
- ✨ 自定义提醒时间设置
- ✨ 已完成记录查看与清空
- ✨ 群聊中每人独立待办列表
- ✨ WebUI 可视化配置

### 支持的指令

| 指令 | 说明 |
|------|------|
| `/todo add <内容> [时间]` | 添加待办 |
| `/todo list` | 查看未完成待办 |
| `/todo done <序号>` | 标记完成 |
| `/todo del <序号>` | 删除待办 |
| `/todo history` | 查看已完成记录 |
| `/todo clear` | 清空已完成记录 |
| `/todo remind <序号> <时间>` | 设置自定义提醒 |
| `/todo help` | 查看帮助 |
